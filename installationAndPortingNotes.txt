% installationAndPortingNotes.txt

Installing on Windows:

    Serial port named COM10
    Note different video driver issues
    Requires SFTP client (WinSCP)

Installing on Linux;

    Serial port named /dev/ttyACM0
        add: -Dgnu.io.rxtx.SerialPorts=/dev/ttyACM0:/dev/ttyACM1
            to the file: /usr/local/MATLAB/R2012b/bin/glnxa64/java.opts
    Note different video driver names and issues
    SFTP client different
    Nb. Requires sharing a public key with the server
